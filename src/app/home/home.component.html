Welcome to the autocomplete-dropdown example page!

So, what is it?

Autocomplete Multidropdown is an Angular Material component designed to allow a user to use an autocomplete text field with a Material multiselect list. This allows the user to search for, and select, multiple values from a list of data. This library is built using Angular Reactive Forms, allowing elements to be created and managed the same way you typically manage reactive forms.
